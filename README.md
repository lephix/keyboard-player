# Keyboard Player
这是一个键盘练习的程序，用游戏的方式为用户提供键盘练习能力。用户在使用这个练习程序的同时，还可以学习英文拼写，古诗词复习等能力。

# 核心玩法
用户能看到一篇上若干单词或者若干汉字的文稿，分为多行展示在用户面前。每一行之间都有较大的空白之处，这里是用户的输入区域，用户必须在此区域进行输入，目标是完全和上面那一行范文的内容精准一样，包括标点符号，空格，半角/全角符号等。

用户某一篇的完成时间和每分钟键盘敲击次数是关键记录指标。

## 语言、年级和难度选择
用户在进入练习之前要求先选择对应的语言（中文/英文），年级（小学、初中、高中），以及难度（简单、困难），选择之后从题库内随机选择一篇范文。

范文库内会有标记好的不同语言的范文，并且标记好了年级。

难度这个概念是在用户参与游戏的时候，是否会有范文中的个别单词、字符被隐藏，需要用户根据自己的认知来推测结果。


## 情绪价值
希望鼓励用户的锻炼，需要增加及时的正反馈，当用户输入正确的内容，可以获得声音、颜色等方面的鼓励，当打啵记录时有明显的提醒。
