# Keyboard Player 业务需求文档

## 1. 产品概述

键盘练习程序，以卡通像素风格的游戏方式为用户提供键盘练习能力。
用户在练习的同时，可以学习英文拼写、古诗词复习等。

## 2. 目标用户

需要练习键盘输入的学生群体（小学、初中、高中）。

## 3. 核心玩法

### 3.1 范文展示与输入

- 用户看到一篇范文，由程序自动分行展示
- 每行范文下方留有输入区域，带有闪烁光标提示当前输入位置
- 用户在输入区域逐行输入，目标是精确匹配范文内容
- 匹配要求包括：标点符号、空格、半角/全角符号完全一致
- 中文输入通过系统输入法完成

### 3.2 练习前选择

- 语言选择：中文 / 英文
- 年级选择：小学 / 初中 / 高中
- 难度选择：简单 / 困难
- 选择完成后从范文库中随机抽取一篇范文
- 全流程支持键盘操作：方向键切换选项、Enter/Space 确认、Esc 返回上一步

### 3.3 困难模式

- 英文：最多 1/3 的字母被隐藏，每个单词最多隐藏 1 个字符
- 中文诗词：每句话最多隐藏 1 个字
- 被隐藏的字符以像素风格图标展示（如问号方块等）
- 用户仍需正确输入被隐藏的内容
- 隐藏规则支持后续调整

### 3.4 关键指标

- 单篇完成时间
- 每分钟键盘敲击次数（KPM）

### 3.5 暂停与退出

- 打字练习中按 Esc 键可暂停游戏
- 暂停时显示半透明遮罩，游戏计时冻结
- 暂停状态下可选择：继续练习（Enter/Space/Esc）或退出到主菜单（Q）
- 中途退出不记录任何成绩

## 4. 情绪价值 / 正反馈系统

### 4.1 实时反馈

- 正确输入：颜色变化（绿色高亮）+ 正反馈音效
- 错误输入：颜色提示（红色）+ 轻微提示音

### 4.2 破纪录提醒

- 刷新个人最佳成绩时，有明显的庆祝效果（音效 + 视觉）

## 5. 数据记录

- 单机单用户，无需登录
- 练习记录（完成时间、KPM、最佳成绩）保存至本地文件

## 6. 范文库规范

### 6.1 存储目录

所有范文统一存放在 `assets/texts/` 目录下，平铺存放，不分子目录。

### 6.2 文件命名格式

```
{语言}_{年级}_{序号}.json
```

- 语言：`en`（英文）、`zh`（中文）
- 年级：`elementary`（小学）、`middle`（初中）、`high`（高中）
- 序号：三位数字，从 001 开始递增
- 示例：`en_elementary_001.json`、`zh_high_003.json`

### 6.3 单篇范文 JSON 结构

```json
{
  "id": "en_elementary_001",
  "language": "en",
  "grade": "elementary",
  "title": "My First Day",
  "author": "Anonymous",
  "content": "The sun was shining brightly. I walked to school with my bag. My teacher smiled and said hello."
}
```

- `id`：与文件名一致（不含 .json 后缀）
- `language`：`en` 或 `zh`
- `grade`：`elementary` / `middle` / `high`
- `title`：范文标题
- `author`：作者（可选，默认 "Anonymous"）
- `content`：范文全文，为一整段文本，由程序根据窗口宽度自动分行

### 6.4 新增范文

后续新增范文只需按照上述命名格式和 JSON 结构放入 `assets/texts/` 目录即可，程序启动时自动扫描加载。

## 7. 视觉风格

- 整体采用卡通像素风格
- 护眼深色主题
